# Requirement: Quiz module

**What:** A 10-question applied knowledge check covering all 8 prior modules, written from a data-engineering angle (e.g. "which field would break this join," not "define a bond").

**Why:** The learner asked for an interactive, not purely reading-based, experience. A quiz forces active recall and surfaces which modules need re-reading, and gives a concrete sense of progress beyond just "marked as done."

**Not:** Not a certification or scored assessment with a pass/fail bar — it's a self-check.

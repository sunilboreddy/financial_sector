# Requirement: Finance data pipelines module

**What:** Cover the recurring pipeline patterns in finance data engineering — position reconciliation, EOD batch processing, corporate actions, regulatory report generation — plus a tools comparison table.

**Why:** This is where all prior modules (instruments, markets, data rules, risk, compliance) come together into the shape of actual day-to-day DE work. It's intended as the "so this is the job" capstone module before the quiz.

**Not:** Does not teach any specific tool in depth — the tools table is a comparison/orientation aid, not a tutorial.
